## **Step 3: Create Firewall Zones**
1. Create the **Internal Zone**:
   ```
   configure terminal
   zone security IN-ZONE
   exit
   ```
2. Create the **External Zone**:
   ```
   zone security OUT-ZONE
   exit
   ```

---

## **Step 4: Define Traffic to Be Inspected**
1. Create an **Access Control List (ACL)** for internal traffic:
   ```
   access-list 101 permit ip 192.168.3.0 0.0.0.255 any
   ```
2. Create a **Class Map** referencing the ACL:
   ```
   class-map type inspect match-all IN-NET-CLASS-MAP
   match access-group 101
   exit
   ```

---

## **Step 5: Define Firewall Policies**
1. Create a **Policy Map**:
   ```
   policy-map type inspect IN-2-OUT-PMAP
   ```
2. Attach the **Class Map** to the policy:
   ```
   class type inspect IN-NET-CLASS-MAP
   ```
3. Enable traffic **inspection**:
   ```
   inspect
   exit
   exit
   ```

---

## **Step 6: Apply the Firewall Policy**
1. Create a **Zone Pair**:
   ```
   zone-pair security IN-2-OUT-ZPAIR source IN-ZONE destination OUT-ZONE
   ```
2. Attach the **Policy Map**:
   ```
   service-policy type inspect IN-2-OUT-PMAP
   exit
   ```

---

## **Step 7: Assign Interfaces to Zones**
1. Assign **G0/1 (LAN) to IN-ZONE**:
   ```
   interface g0/1
   zone-member security IN-ZONE
   exit
   ```
2. Assign **S0/0/1 (WAN) to OUT-ZONE**:
   ```
   interface s0/0/1
   zone-member security OUT-ZONE
   exit
   ```

---
